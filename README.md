# Usage index [options]

| -h | --help   | output usage information |
|----|----------|--------------------------|
| -u | --url    | The youtube url          |
| -o | --output | Output dir               |
| -n | --name   | Output name              |


# Installing
```
sudo npm i youli -g 
```

# Example using

```
youli -u https://www.youtube.com/watch?v=9vz4bh0KYGc -o ~cagatay/Desktop -n tavuklariPisirmisem
```
